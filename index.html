A Few Tags

  • Block Level Tags
    • These elements have an invisible wrapper around them that can be manipulated using CSS and JavaScript
    • Paragraph
    • Div
  • Inline Tags
    • These tags have no such manipulatible wrapper and are found "in the line" of the code itself
    • Anchor (href)
    • Strong/Bold
    • Emphasis
    • Break
    • Span

Computers Are Stupid
The code has to be written exactly the way a computer can understand then or things can go horribly wrong. This is known as correct syntax.
